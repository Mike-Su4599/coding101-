# Vegetarian Restaurant Site Selection Analysis

## Project Overview
This project analyzes suitable locations in Taichung for opening vegetarian restaurants by integrating demographic, competitor, and geographic data.

## Business Problem
How can population structure, competitor density, and geographic information be combined to support restaurant site selection?

## My Role
I was responsible for:
- scraping competitor business information
- cleaning address data
- converting addresses into coordinates
- organizing geographic data
- supporting the site selection logic used in the final analysis

## Data Sources
- competitor restaurant information
- demographic data
- geographic and administrative area data

## Methods
- Web scraping with Python
- Address cleaning and geocoding
- Data integration across multiple sources
- Site selection logic based on population and competitor distribution

## Key Contributions
- Integrated more than 20,000 records of demographic, restaurant, and geographic data
- Converted scattered external data into a usable site selection framework
- Supported the final analysis for identifying suitable districts in Taichung

## Result
The project won the **Best Popularity Award** in the final round of the Coding101 competition.
